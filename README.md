# EEZO Target Optimization Model

## 概要

新日本海商事のECサイト「EEZO」のBtoC事業において、フェリーユーザーをベースとしたターゲット設定と商品戦略を数理モデルで最適化するプロジェクト。

## 検証する問い

1. **どのセグメントに注力すべきか？**
   - S1（超富裕層）、S2（富裕層）、S3（準富裕層）

2. **最適な価格帯構成は？**
   - P1（〜5千円）〜 P4（2万円〜）

3. **目的関数による最適解の違いは？**
   - 売上最大化 vs 粗利最大化 vs LTV最大化
